CloudFlow

跨平台 Linux 实时数据监控面板，基于 Dart 构建高性能后端，支持自定义仪表盘与告警规则，强调一致体验。

详细介绍 下载 GitHub
/ 功能特性

核心能力

实时洞察系统状态，掌控每一刻

实时数据监控

支持 CPU、内存、磁盘、网络等系统指标的实时采集与可视化，毫秒级数据刷新。

自定义仪表盘

灵活的可视化布局系统，支持拖拽式仪表盘编辑，自由组合图表、指标卡和状态面板。

智能告警规则

基于阈值和趋势的告警引擎，支持多条件组合、通知渠道（邮件、Webhook）及升级策略。

跨平台一致体验

基于 Dart/Flutter 构建，在桌面和 Web 端提供一致的用户界面与交互体验。
