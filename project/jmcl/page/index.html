JMCL

轻量级跨平台 Minecraft 启动器，基于 Java 构建，支持多版本管理、Mod 加载与一键启动，提供简洁高效的游戏体验。

详细介绍 下载 GitHub
/ 功能特性

核心能力

让 Minecraft 启动更简单、更自由

多版本管理

支持从 1.7 到最新版的 Minecraft 全版本管理，自动下载安装，在多个版本间无缝切换。

Mod 加载管理

集成 Forge、Fabric 等主流 Mod 加载器，支持 Mod 可视化安装、启用/禁用和一键更新。

跨平台支持

基于 Java 构建，原生支持 Windows、macOS 和 Linux，提供一致的用户界面和操作体验。

一键启动

智能版本检测与依赖自动补全，配置完成后一键启动游戏，无需手动配置 Java 参数。
