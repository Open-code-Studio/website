OpenKit.Core

TypeScript 生态的工具链核心库，提供 CLI 脚手架、配置解析、日志系统等基础能力，为其他项目提供底层支持。

详细介绍 下载 GitHub
/ 功能特性

核心能力

为 TypeScript 生态提供坚实底层

CLI 脚手架

提供快速项目初始化的命令行工具，支持自定义模板、交互式配置生成，提升开发启动效率。

配置解析

统一的多格式配置解析引擎，支持 JSON、YAML、TOML、环境变量等，提供类型安全的配置访问。

日志系统

高性能结构化日志框架，支持多级别、多输出目标、自定义格式化，满足生产环境日志需求。

插件机制

灵活的插件架构，支持生命周期钩子、依赖注入，方便扩展和定制工具链行为。
